- # Classical

- **Linear vs Nonlinear System Characteristics:** [Nonlinear Systems Notes](https://www.ece.ucsb.edu/~hespanha/ece229/)
  - Summary: Explains differences between linear and nonlinear systems, including superposition, nonlinear gain behavior, and asymmetric dynamics.

- **Euler Integration vs Runge-Kutta Methods:** [MIT Numerical Methods Notes](https://web.mit.edu/10.001/Web/Course_Notes/Differential_Equations_Notes/node3.html)
  - Summary: Discusses numerical integration methods for nonlinear systems and motivates adaptive solvers like Runge-Kutta.

- **Control Lyapunov & Barrier Functions (CLF/CBF):** [IEEE Paper](https://www.ieee-jas.net/en/article/doi/10.1109/JAS.2023.123075)
  - Summary: Introduces Lyapunov-based nonlinear stability analysis and safety constraints.

- **Lyapunov Stability Definitions:** [Stanford Lecture Notes](https://see.stanford.edu/materials/lsoeldsee263/transcripts/IntroToLinearDynamicalSystems-Lecture15.html)
  - Summary: Covers stability, asymptotic stability, and global asymptotic stability concepts.

- **Equilibrium Points in Nonlinear Systems:** [Khalil Nonlinear Systems Book](https://www.pearson.com/en-us/subject-catalog/p/nonlinear-systems/P200000003479/9780130673893)
  - Summary: Discusses equilibrium analysis and nonlinear system behavior near operating points.

- **Phase Plane Analysis:** [MIT Phase Portrait Tutorial](https://ocw.mit.edu/courses/16-30-feedback-control-systems-fall-2010/resources/mit16_30f10_lec22/)
  - Summary: Demonstrates trajectory behavior, basins of attraction, and nonlinear phase portrait analysis.

- **Limit Cycles and Oscillations:** [Van der Pol Oscillator Notes](https://mathworld.wolfram.com/vanderPolEquation.html)
  - Summary: Explains stable limit cycles and nonlinear oscillatory behavior.

- **Jacobian Linearization Method:** [Control Theory Notes](https://underactuated.mit.edu/lyapunov.html)
  - Summary: Uses Jacobian matrices and eigenvalue analysis to determine local nonlinear stability.

- **Lyapunov Direct Method:** [MIT Underactuated Robotics](https://underactuated.mit.edu/lyapunov.html)
  - Summary: Introduces Lyapunov functions to analyze nonlinear system stability.

- **Radial Unboundedness and Global Stability:** [Nonlinear Control Lecture Notes](https://faculty.washington.edu/chx/teaching/advcontrol2/dobnotes.pdf)
  - Summary: Explains conditions required for global asymptotic stability.

- **LaSalle Invariance Principle:** [Math Stack Exchange Overview](https://math.stackexchange.com/questions/564369/lasalles-invariance-principle)
  - Summary: Introduces invariant sets and convergence analysis for nonlinear systems.

- **Lienard Theorem and Stable Limit Cycles:** [Scholarpedia Article](http://www.scholarpedia.org/article/Lienard_equation)
  - Summary: Gives conditions for existence of unique stable limit cycles.

- **Event-Based and Quantized Nonlinear Control:** [Springer Article](https://link.springer.com/article/10.1007/s11633-015-0906-x)
  - Summary: Presents event-triggered nonlinear control methods for computational efficiency.

- **Event-Triggered Control of Nonlinear Stochastic Systems:** [DOI Link](https://doi.org/10.1109/tac.2019.2916285)
  - Summary: Addresses nonlinear stochastic systems with reduced control update strategies.

- **Nonlinear Model Predictive Control (MPC):** [DOI Link](https://doi.org/10.1109/tac.2018.2800789)
  - Summary: Optimization-based nonlinear control with constraints and predictive stability.
