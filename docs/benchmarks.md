# Benchmark Systems

## 1. Cart-Pole (Inverted Pendulum)
- Widely used
- Tests stabilization and swing-up

## 2. Nonlinear Mass-Spring-Damper
- Good for studying oscillatory systems
- Can include nonlinear damping

## 3. Pendulum (Gym)
- Simpler than cart-pole
- Continuous control

## 4. Other Options
- Quadrotor dynamics
- Lorenz system (chaotic)
- Robotic manipulators

## Evaluation Metrics
- Tracking error (MSE, RMSE)
- Control effort
- Settling time
- Overshoot
- Robustness to disturbances

## Experimental Protocol
- Same initial conditions
- Same disturbance profile
- Same simulation time
