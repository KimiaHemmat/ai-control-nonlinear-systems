# Taxonomy of Control Approaches

## 1. Classical Control
- PID
- LQR / LQG
- Model Predictive Control (MPC)
- Sliding Mode Control
- Feedback Linearization

## 2. ML-Augmented Control
- Bayesian Optimization for tuning
- Evolutionary algorithms
- Residual learning (model correction)

## 3. Classical Machine Learning
- Support Vector Machines (SVM/SVR)
- Gaussian Processes
- Sparse regression (SINDy)

## 4. Deep Learning Control
- Multi-layer perceptrons (MLP)
- Recurrent neural networks (RNN/LSTM)
- Neural ODEs

## 5. Reinforcement Learning
- Model-free (PPO, SAC, DQN)
- Model-based RL
- Safe RL

## 6. Hybrid Approaches
- MPC + Neural Networks
- Physics-informed neural networks (PINNs)
- Learning-based observers

## Notes
- Categories may overlap
- Hybrid methods are increasingly dominant
