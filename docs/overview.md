# Overview

This repository explores the use of Artificial Intelligence (AI) in controlling nonlinear dynamic systems, with a focus on comparing classical and learning-based approaches.

## Motivation
Nonlinear systems are difficult to control due to:
- Nonlinearity and instability
- Model uncertainty
- External disturbances

Classical control methods provide strong guarantees but rely on accurate models. AI-based methods offer adaptability but often lack stability guarantees.

## Scope
This repository focuses on:
- Classical control (PID, MPC, etc.)
- Machine learning–augmented control
- Data-driven system identification
- Deep learning and reinforcement learning controllers

## Research Questions
- How do AI-based controllers compare to classical methods?
- When does learning improve control performance?
- What are the trade-offs in stability, robustness, and interpretability?
